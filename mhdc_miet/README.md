# Mental Health Disorder Classification Project

## Overview

This project aims to classify mental health disorders based on various features using machine learning techniques. The classification model is trained on a dataset containing information about patients and their mental health conditions.

## Installation

1. **Clone Repository**: Clone this repository to your local machine:

git clone https://github.com/rayzcell/mental-health-disorder-classification.git


2. **Navigate to Directory**: Navigate to the project directory:

cd mental-health-disorder-classification


3. **Create Virtual Environment** (Optional): Create a virtual environment (recommended):

python -m venv venv


4. **Activate Virtual Environment**: Activate the virtual environment:
- On Windows:
  ```
  venv\Scripts\activate
  ```
- On macOS/Linux:
  ```
  source venv/bin/activate
  ```

5. **Install Dependencies**: Install the required dependencies:
    pip install -r requirements.txt


## Usage

To run the Streamlit application, execute the following command in your terminal:

streamlit run app.py


This will start the Streamlit server, and you can access the application in your web browser at `http://localhost:8501`.
