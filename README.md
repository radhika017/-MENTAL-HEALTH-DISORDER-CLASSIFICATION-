# Mental Health Disorder Classification (MHDC) 🧠💙

## 📌 Project Overview
Mental Health Disorder Classification (MHDC) is a Machine Learning based project developed to classify different mental health disorders using various supervised learning algorithms and ensemble techniques.

The project focuses on identifying mental health conditions such as:
- Normal
- Depression
- Bipolar Type-1
- Bipolar Type-2

using symptom-based datasets and machine learning models.

---

## 🎯 Objective
The main objective of this project is to analyze mental health data and classify disorders accurately using Machine Learning techniques to support early identification and awareness.

---

## 🚀 Features
✅ Mental health disorder classification  
✅ Data preprocessing and analysis  
✅ Ensemble machine learning model  
✅ Prediction and evaluation system  
✅ Performance analysis using ML metrics  
✅ Classification of multiple disorder categories
